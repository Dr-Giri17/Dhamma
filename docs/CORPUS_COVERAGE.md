# Corpus coverage

This repository exposes the complete navigation structure of the three Tipiṭaka baskets, but it does **not** contain the full Tipiṭaka text.

## Exact local coverage

| Language | Imported editions | Coverage |
|---|---:|---|
| Pāli | 8 | MN 10, MN 118, DN 31, AN 3.65, SN 56.11, Snp 1.8, Snp 2.1, Snp 2.4 |
| English | 9 | The same eight suttas plus Dhammapada (423 verses) |
| Russian | 5 | MN 10, MN 118, DN 31, SN 56.11, Snp 1.8 |
| Indonesian | 0 | No matching published CC0 Theravāda Pāli seed edition at the pinned revision |

There are 9 imported texts, 22 imported editions, and 1,437 normalized segments.

## Catalog-only coverage

- Vinaya Piṭaka: structure only.
- Sutta Piṭaka: major collections shown; exact local texts listed above.
- Abhidhamma Piṭaka: seven-book structure only.
- Visuddhimagga: post-canonical metadata/source link only; no local text.

## Reader behavior

The interface language and text edition are separate. Unavailable RU or ID requests are never relabeled; the reader explicitly states that the requested translation is absent and identifies any English fallback. Pāli/translation parallel reading is available for the eight aligned Bilara suttas.
